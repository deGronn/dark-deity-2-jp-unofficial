# Dark Deity 2 非公式日本語化パッチ

Dark Deity 2 を日本語化することを目的とした非公式パッチです。  
本パッチは個人によるファン制作であり、公式とは一切関係ありません。  
ゲーム本体は含まれていません。使用には別途 Dark Deity 2 本体が必要です。

また、本パッチはすでに投稿されている他のパッチとは、制作主・内容ともに無関係の別物です。

2026/04/04 のアップデート時点のバージョンで正常に動作することを確認しています。  
ただし、今後のアップデートにより正常に動作しなくなる可能性があります。

## 導入方法
1. まずゲームを一度起動し、正常に起動することを確認してください。
2. ゲームを終了してください。
3. ファイルをダウンロードして解凍してください。
4. 解凍したファイルをすべて `Steam\steamapps\common\Dark Deity 2` に入れて、上書きしてください。
 ※分からない方はSteamのライブラリのDark Deity 2を右クリックし、管理タブから「ローカルファイルを閲覧」を選択してください。
6. ゲームを起動し、タイトル画面右下に「日本語（非公式）」が追加されていれば導入成功です。
7. 該当部分を押すと、日本語に切り替わります。
