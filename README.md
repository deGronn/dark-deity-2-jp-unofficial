# Dark Deity 2 非公式日本語化パッチ

Dark Deity 2 を日本語化することを目的とした非公式パッチです。  
本パッチは個人によるファン制作であり、公式とは一切関係ありません。  
ゲーム本体は含まれていません。使用には別途 Dark Deity 2 本体が必要です。  
また、すでに公開されている他のパッチとは無関係の別物です。

2026/04/04 のアップデート時点のバージョンで正常に動作することを確認しています。  
ただし、今後のアップデートにより正常に動作しなくなる可能性があります。

## 導入方法

1. ゲームを一度起動し、正常に起動することを確認してください。  
2. ゲームを終了してください。  
3. 配布ファイルをダウンロードして解凍してください。  
4. 解凍したフォルダを開き、中に入っているファイルをすべて Dark Deity 2 のゲームフォルダに直接入れてください。  
   ※ 入れるのは `apply_patch.bat`、`hpatchz.exe`、`patch.ddp`、`extra`（フォルダ）の4つです。  
5. ゲームフォルダ内にある `apply_patch.bat` をダブルクリックして実行してください。  
6. 黒い画面が開き、自動でパッチ適用が始まります。  
7. ゲームを起動し、タイトル画面右下に「日本語（非公式）」が追加されていれば導入成功です。  
8. 該当部分を押すと、日本語に切り替わります。  

### ゲームフォルダの場所

Dark Deity 2 のゲームフォルダは、Steam の標準設定では以下の場所にあります。

`C:\Program Files (x86)\Steam\steamapps\common\Dark Deity 2`

インストール先が分からない場合は、Steam ライブラリで Dark Deity 2 を右クリックし、  
**「管理」→「ローカルファイルを閲覧」** を選ぶとゲームフォルダを開けます。
