Dark Deity 2 を日本語化することを目的とした非公式パッチです。  
本パッチは個人によるファン制作であり、公式とは一切関係ありません。

また、本パッチはすでに投稿されている他のパッチとは、制作主・内容ともに無関係の別物です。

2026/04/04 のアップデート時点のバージョンで正常に動作することを確認しています。  
ただし、今後のアップデートにより正常に動作しなくなる可能性があります。

## 導入方法
1. ファイルをダウンロードして解凍してください。
2. 解凍したファイルをすべて `Steam\steamapps\common\Dark Deity 2` に入れて、上書きしてください。
